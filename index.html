Fusion 360 GCODE feed rate restriction removal

Autodesk say "When using Fusion 360 for Personal Use, the feedrate of rapid moves is reduced to match the feedrate of cutting moves, which can increase machining time."
This utility removes that restriction from a GCODE file that was generated by Fusion 360

Enter the retract height for the tool paths.
Select your GCODE file from Fusion 360.
Hit the button to copy new GCODE
